# 🛰️ Academic Morning Post (PaperBot)

> **「不再被文獻閱讀折磨，釋放你的靈魂。去健身、去跑步、去享受生活。」**

這是一個專為研究生打造的**自動化學術論文追蹤與轉譯系統**。我們利用爬蟲技術與 AI，將艱深的 MDPI 論文轉化為質感的「左右翻頁式」白話海報，讓你在學術生活中順利度過，把時間還給健康。

---

## ✨ 核心理念：學術與健康的平衡

身為研究生，不應該淹沒在無盡的 PDF 檔案與晦澀的學術黑話中。
* **釋放時間**：自動化執行枯燥的追蹤工作，讓你去健身、跑步、擁抱陽光。
* **直觀理解**：將論文翻譯成「白話文」，每天花 3 分鐘看一篇，掌握學術前沿。
* **完全免費**：基於開源技術，使你的學術生涯無所不能。

---

## 🛠️ 技術架構

本專案深度整合了現代化自動化流程：
* **OpenClaw 技術**：賦予系統強大的邏輯執行與自動化任務能力。
* **智能爬蟲**：自動從 **MDPI Remote Sensing** 抓取最新論文網址。
* **AI 深度轉譯**：將複雜的論文結構轉化為「一句話核心」、「驚人發現」等易讀格式。
* **動態網頁渲染**：自動生成支援左右翻頁（Swiper.js）的高質感行動端網頁。
* **即時發布**：一鍵部署至 **GitHub Pages**。

---

## 📖 執行流程 (Workflow)

系統透過 `paperbot.py` 的不同模式進行三階段自動化：

### 1. 採集階段 (Collect)
python paperbot.py --mode collect
自動掃描網站，對比歷史記錄，產出 temp_task.md。

### 2. 轉譯階段 (AI Translation)
將 temp_task.md 交給 AI，並將產出的結果儲存為 temp_result.md。

### 3. 合併與發布 (Merge & Auto-Render)
python paperbot.py --mode merge
一鍵執行三合一任務：

歸檔：將 AI 結果存入長期數據庫 paper_summary.md。

渲染：更新翻頁網頁 index.html（內文字體統一 1.3rem + 紅色粗體重點）。

推送：自動 Git Push，完成 GitHub Pages 更新。
